Client SDK Integration

Each app embeds client_sdk/license_core.py from this repo. Set the constants at the top of the file for each app:

APP_NAME = "TEEMOX_MAILER" # or INFOMANIAK_API / SHOPIFY_API APP_DIR_NAME = "TeemoxMailer" # %APPDATA% subdirectory GITHUB_OWNER = "anupamnath" GITHUB_REPO = "teemox-licensing"

Security Checklist

  • ✅ Ed25519 private key stored ONLY in GitHub Actions Secret (LICENSE_PRIVATE_KEY_PEM)
  • ✅ Public key embedded in each app binary — tamper-evident
  • ✅ Machine-locked licenses prevent key sharing
  • ✅ Revocation list at public/valid_licenses.json enforced on every sync
  • ✅ Encrypted local sync cache with anti-rollback (clock manipulation protection)
  • ✅ Grace period prevents abuse of offline mode
  • ✅ PAT never leaves your browser (sessionStorage only)
